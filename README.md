#  HorrposShooter

> A retro-inspired first-person shooter built with **Python** and **Pygame**.

HorrposShooter is a classic FPS inspired by games like **DOOM**. Fight hostile enemies, explore maze-like levels, and survive using your shotgun in a fast-paced raycasting world.

---

## Features

*  Classic DOOM-style raycasting engine
*  Animated shotgun with shooting and reloading
*  Enemy AI with pathfinding
*  Mouse aiming and WASD movement
*  Player health system
*  Damage effects
*  Sound effects and background music
*  Victory screen
*  Game Over screen
*  Custom maps and textures

## Controls

| Key            | Action        |
| -------------- | ------------- |
| **W**          | Move Forward  |
| **A**          | Move Left     |
| **S**          | Move Backward |
| **D**          | Move Right    |
| **Mouse**      | Look Around   |
| **Left Click** | Fire Weapon   |
| **ESC**        | Exit Game     |
